Программа начинается с подключения модуля random. Он нужен для того, чтобы компьютер мог сгенерировать случайное число. Далее создаётся переменная n, в которую записывается случайное число от 1 до 100 с помощью random.randint(1, 100). Это и есть число, которое должен угадать пользователь.

После этого выводится сообщение print("Угадай число от 1 до 100"), чтобы объяснить пользователю, что нужно делать. Затем запускается цикл for i in range(3), который означает, что у пользователя есть только 3 попытки.

Внутри цикла программа просит ввести число с помощью input("Число: "). Введённое значение сразу преобразуется в число с помощью int(), чтобы его можно было сравнивать.

Дальше идёт проверка. Если введённое число больше загаданного (x > n), программа выводит "меньше", подсказывая, что нужно брать число меньше. Если введённое число меньше (x < n), выводится "больше". Если же числа равны, программа пишет "Угадал" и с помощью break выходит из цикла, потому что игра закончена.

Если пользователь не угадал за 3 попытки, срабатывает блок else после цикла for. Это особенность Python: такой else выполняется только если цикл завершился без break. В этом случае выводится "Не угадал".

В самом конце программа всегда выводит print("Было:", n), чтобы показать, какое число загадал компьютер.
